# federal-budget-analysis
Hello my name is Emmanuel Asinas and here is a project I worked on for my portfolio, I outlined an analysis on the differences between military, education, and health spending in the US for FY 2025. I used Claude to create the code for all of the project, however I was able to prompt Claude in specifying the information I wanted and was able to correct Claude with another prompt in order to get the specific information in order to analyze. 
Listed are the components of my project, and the full interactive dashboard of my project can be found here: https://public.tableau.com/views/FY2025FederalBudgetAnalysisDefenseEducationMedicare/FY2025BudgetAnalysis?:language=en-US&:sid=&:redirect=auth&:display_count=n&:origin=viz_share_link

And the Excel sheet of all the data's components are found in this link: https://docs.google.com/spreadsheets/d/1OwkhXe7Ck0JTUlijMU0nalI28ImeJtVBn2Y5s1BNJHk/edit?usp=sharing

And the data of medicare and medicaid, as well as education budgets were found through a google search of 'Federal spending on Medicare/Medicaid in 2025', As well as a google search of 'Federal funding for education for 2025.'
